Operating Systems


GNU/Linux


It is highly recommended if you are getting into networking or coding to get familiar with GNU/Linux. A lot of servers are hosted on GNU/Linux and GNU/Linux is a lot of people's first choice for development.

Starting out can be scary, there are so many distro's to start from. You see the memes of Linux users downloading everything from terminal, or even loading videos to watch from terminal. A lot of distro's have came to be since the booming interest and popularity have came to be. Many people start out on Ubuntu, then they get a little more into it once they are more familiar. This is when they head to the Ubuntu flavors (Kubuntu, Xubuntu, Lubuntu, etc). Eventually, they'll try Manjaro, then Fedora, and finally end the distro hopping somewhere around Gentoo and Arch Linux before they settle for one Linux distro.

For beginning, I recommend using Ubuntu. It comes pretty stable out of the box and features Gnome (desktop environment). Gnome DE is a great looking, snappy, customizable desktop environment, but if you don't like it, you can easily uninstall it and find something else. Xubuntu is still the same as regular Ubuntu, it features the Xfce DE which is a whole lot lighter DE than Gnome.

When it comes to Ubuntu, it isn't as complicated as most people think to install programs. Ubuntu features it's own Software Centre. Can't find what you're looking for there? Google your application, install the .deb package file and do it that way. Not working? Update your repository, and apt-get install from your terminal. There are many videos on Youtube to help you get started, if you want to dip your feet in before you jump in, install OracleBox VM and try it in a virtual environment for a week. You can also simply make a bootable flash drive and demo Ubuntu off the flash drive without having to format your system.

Code Editors and IDEs for GNU/Linux


The difference between a code editor and an IDE (integrated development environment) is that a code editor is a text editing interface without the advanced features that IDEs come with. IDEs are a software development environment that features tools such as a debuggers and compilers. For code editors, I highly recommend Visual Studio Code and Atom. For IDEs, IntelliJ IDEA (for Java and Android devs) and, Eclipse (Java devs and anyone who codes in C, C++, COOL, JavaScript, PHP, Perl, Python).
