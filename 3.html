Hammer Editor


The Valve Hammer Editor (more informally known as Hammer, and previously called Worldcraft) is the official mapping tool for the Goldsource and Source engines (which most newer Valve games run under). It is also included in every game made with Source Engine that is not a mod


On this page, I've linked a tutorial and a website to get started on mapping for Source engine games.
